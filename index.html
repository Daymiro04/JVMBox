Características

  • ⚙️ 100% portable, sin instalación
  • 🖥️ Interfaz gráfica en Java con splash screen
  • 💾 Soporte de ISOs y disco virtual `.qcow2`
  • 🎧 Audio, video, arranque desde CD o disco
  • 📦 Compatible con Debian, Ubuntu, TinyCore

Cómo instalar JVMBox

  1. Descarga el archivo JVMBox.zip
  2. Extrae la carpeta completa
  3. Asegúrate de tener Java 8+ instalado (puedes usar versión portable)
  4. Haz doble clic en JVMBox.jar o ejecuta: java -jar JVMBox.jar
  5. Selecciona un ISO si deseas instalar un sistema
  6. ¡Listo! Tu Linux se ejecutará en una VM dentro de tu PC

Capturas

Preguntas frecuentes

  • ¿Necesito instalar Java?
    Sí, al menos Java 8. Puedes usar una versión portable.
  • ¿Funciona sin permisos de administrador?
    Sí, JVMBox y QEMU funcionan 100% sin instalar nada.
  • ¿Qué sistemas puedo ejecutar?
    TinyCore, Debian, Ubuntu y otros compatibles con x86_64.
  • ¿Se guarda el progreso?
    Sí, usando el disco `hdd.qcow2`, puedes guardar tus datos.
